# Wireshark Lab: Packet Analysis Basics

## Objective
Learn to capture and analyze network traffic using Wireshark.

## Steps
1. Install Wireshark
2. Capture HTTP traffic
3. Filter for specific protocols
4. Follow TCP streams
5. Extract files from captures

## Commands & Filters
- `http` - Show only HTTP traffic
- `tcp.port == 443` - HTTPS traffic
- `ip.addr == 192.168.1.1` - Specific IP
