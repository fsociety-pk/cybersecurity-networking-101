## What's Inside

This repository covers networking essentials for cybersecurity:

- **Fundamentals**: Network types, devices, IP addressing, subnetting
- **Protocol Analysis**: TCP/IP models, ports, protocols, encryption
- **Practical Tools**: Wireshark, Nmap, packet analysis techniques
- **Quick References**: Cheatsheets for ports, subnetting, filters
- **Lab Exercises**: Hands-on packet analysis and network troubleshooting
